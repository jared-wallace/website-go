# Requirements: website-go

**Defined:** 2026-03-28
**Core Value:** A reader visits jared-wallace.com and reads well-rendered markdown blog posts with images in a distinctive, memorable design.

## v1.1 Requirements

Requirements for The Wild Meridian milestone. Each maps to roadmap phases.

### Branding

- [ ] **BRAND-01**: Blog title reads "The Wild Meridian" on all public pages (header, browser tab, etc.)
- [ ] **BRAND-02**: RSS feed title and metadata reflect "The Wild Meridian"
- [ ] **BRAND-03**: XML sitemap, Open Graph meta tags, and any SEO surfaces reference "The Wild Meridian"
- [ ] **BRAND-04**: Copyright footer reads "2026 Jared Wallace"

### About

- [ ] **ABOUT-01**: User can navigate to an about page from the main site navigation
- [ ] **ABOUT-02**: About page renders content from a static markdown file on disk
- [ ] **ABOUT-03**: About page matches the existing nautical design (header, footer, dark mode)

### Navigation

- [ ] **NAV-01**: RSS feed is discoverable via a discreet link/icon in the nav or footer
- [ ] **NAV-02**: HTML head includes `<link rel="alternate" type="application/rss+xml">` for autodiscovery

## Future Requirements

None identified for this milestone.

## Out of Scope

| Feature | Reason |
|---------|--------|
| Dynamic about page (DB-backed) | Static markdown is sufficient and easier to maintain |
| Subtitle change | "Dispatches from the deep end" stays as-is |
| Full navigation redesign | Only adding about link and RSS icon |
