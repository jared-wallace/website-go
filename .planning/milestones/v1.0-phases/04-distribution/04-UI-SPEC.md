# Phase 4 — UI Design Contract

> Visual and interaction contract for Phase 4: Distribution.
> Generated by gsd-ui-researcher. Verified by gsd-ui-checker.
>
> **Source note:** This phase adds no new pages. All four features (RSS, OG tags, sitemap, thumbs-up) modify
> existing templates or add machine-readable endpoints. The contract below governs the single new reader-visible
> UI element (the thumbs-up reaction bar on the post page) and the meta/head additions to base.html.

## Spacing Scale

Declared values (all multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, tag padding, inline code padding |
| sm | 8px | Compact spacing: post-meta margin, toc-item margin, tag-pill gap |
| md | 16px | Default element spacing: section padding, blockquote indent, list indent |
| lg | 24px | Section padding: nav padding, card padding, container horizontal padding |
| xl | 32px | Layout gaps: error-page padding, post-body h2 top margin |
| 2xl | 48px | Major section breaks: empty-state vertical padding, pagination top margin |
| 3xl | 64px | Page-level spacing: container vertical padding |

Exceptions:
- Touch target minimum: 44px applied to `.dark-toggle`, `.toc-toggle`, `.pagination a`. The thumbs-up button must also meet this minimum: `min-height: 44px` on `.reaction-btn`.
- Reaction bar: padding 8px 16px inside the button; 32px top margin above `.reaction-bar` to separate from post body.

## Component Inventory

New components added in this phase (all in `web/static/main.css`):

### `.reaction-bar`
Container below `.post-body`, above `<footer>`.

```css
.reaction-bar {
  margin-top: 32px;

  justify-content: center;
  padding: 16px 0;

```

### `.reaction-btn`
The thumbs-up button. Transitions from outline (unactivated) to filled (activated).

```css
.reaction-btn {
  background: none;
  border: 2px solid var(--color-accent);
  border-radius: 2rem;
  padding: 8px 16px;
  cursor: pointer;
  font-size: 18px;
  display: inline-flex;
  align-items: center;
  gap: 8px;
  min-height: 44px;
  color: var(--color-text);
  transition: background 0.2s, color 0.2s;
  font-family: var(--font-body);
}

.reaction-btn.reacted,
.reaction-btn:disabled {
  background: var(--color-accent);
  color: var(--color-bg);
  cursor: default;
}
```

### `.reaction-icon` bounce animation
One-shot animation on tap. Class `bounce` added by JS after fetch resolves; removed on `animationend`.

```css
.reaction-btn.bounce .reaction-icon {
  animation: reaction-bounce 0.4s ease;
}

@keyframes reaction-bounce {
  0%   { transform: scale(1); }
  40%  { transform: scale(1.4); }
  70%  { transform: scale(0.9); }
  100% { transform: scale(1); }
}
```

## Interaction Contract

### Thumbs-Up Flow

| State | Visual | Trigger |
|-------|--------|---------|
| Default (not reacted) | Button: transparent fill, `--color-accent` 2px border, emoji + count | Page load; localStorage key absent |
| Disabled pre-react (localStorage guard) | Button: `.reacted` class applied; accent background, muted cursor | localStorage key present on page load |
| Tap (optimistic disable) | Button immediately disabled; prevents double-fire | `click` event — BEFORE fetch resolves |
| Success | Count increments; `.reacted` + `.bounce` classes added; `aria-label` updated to "You gave a thumbs-up" | fetch `.then()` resolves |
| Bounce complete | `.bounce` class removed | `animationend` event on `.reaction-icon` |
| Fetch error | Button re-enabled; no state change; no error copy shown | fetch `.catch()` |

### RSS Auto-Discovery

No reader-visible interaction. The `<link rel="alternate">` tag is picked up by browser chrome and RSS readers automatically. No UI element required.

### Open Graph Tags

No reader-visible interaction. Tags are consumed by crawlers and link-preview renderers. Verified via Twitter Card Validator and Open Graph Debugger (Facebook Sharing Debugger) post-implementation.

## Accessibility Contract

| Requirement | Implementation |
|------------|---------------|
| Reaction button keyboard accessible | `<button>` element; inherits `:focus-visible` outline from global rule (2px solid `--color-accent`) |
| Reaction button touch target | `min-height: 44px` on `.reaction-btn` (matches existing pattern on `.dark-toggle`, `.toc-toggle`) |
| Reaction button screen reader label | `aria-label="Give a thumbs-up"` on initial render; updated to `"You gave a thumbs-up"` by JS after activation |
| Reaction count readable | Plain text numeral in `<span>` — no icon-only count |
| Emoji accessible | `aria-hidden` not required on &#128077; — it is decorative alongside the aria-label on the button |
| RSS link in head | `<link rel="alternate">` — standard machine-readable convention; no ARIA needed |
