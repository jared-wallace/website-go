**1. Create `internal/service/post/feed.go`** with two service methods:

```go
// ListForFeed returns the most recent N published posts for RSS feed generation.
// Reuses the existing repo.ListPublished with offset=0.
func (s *Service) ListForFeed(ctx context.Context, limit int) ([]model.Post, error)
```
This calls `s.repo.ListPublished(ctx, limit, 0)` and returns the raw `[]model.Post` (RSS needs RenderedHTML, not PostSummary).

```go
// ListSlugsForSitemap returns slug + created_at for all published posts.

```
This calls `s.repo.ListPublished(ctx, 10000, 0)` — at blog scale this is fine. No new repo interface method needed.

**2. Create `internal/handler/blog/rss.go`** with:

- `const baseURL = "https://jared-wallace.com"` (module-level constant)
- RSS 2.0 struct types per RESEARCH.md Pattern 2:
  - `RSSFeed` with `XMLName xml.Name`, `Version string`, `Channel RSSChannel`
  - `RSSChannel` with Title, Link, Description, Language, ManagingEditor, LastBuildDate, Items
  - `RSSItem` with Title, Link, Description (type `CDATA`), Author, PubDate, GUID, Categories
  - `RSSGuid` with IsPermaLink bool, Value string
- `CDATA` type implementing `xml.Marshaler` per RESEARCH.md Pitfall 1 — wraps HTML in `<![CDATA[...]]>` to prevent double-escaping
- `buildRSSFeed(posts []model.Post) RSSFeed` function:
  - Channel: Title="The Log", Link=baseURL, Description="dispatches from the deep end", Language="en-us"
  - ManagingEditor: "jaredwallace@jared-wallace.com (Jared Wallace)" (per D-02)
  - LastBuildDate: `time.Now().Format(time.RFC1123Z)`
  - Per item: Title=p.Title, Link=baseURL+"/posts/"+p.Slug, Description=CDATA{p.RenderedHTML}, Author="jaredwallace@jared-wallace.com (Jared Wallace)" (per D-02), PubDate=p.CreatedAt.Format(time.RFC1123Z), GUID={IsPermaLink:true, Value:postURL}, Categories=ParseTags(p.Tags) (per D-03)
- `func (h *BlogHandler) ServeRSS(w http.ResponseWriter, r *http.Request)`:
  - Calls `h.svc.ListForFeed(r.Context(), 25)` (per D-01: most recent 25)
  - Sets Content-Type to "application/rss+xml; charset=utf-8"
  - Writes `xml.Header` via `io.WriteString(w, xml.Header)`
  - Encodes with `xml.NewEncoder(w)` with `enc.Indent("", "  ")`
  - Logs error with slog on encode failure
