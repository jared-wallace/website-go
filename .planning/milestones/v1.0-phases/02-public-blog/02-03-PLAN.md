       - Click any post title, verify it opens at /posts/{slug}

       - Click "[ hide ]" to collapse ToC, click "[ show ]" to expand
       - Verify code blocks have syntax highlighting with accent left border
    4. **Dark mode:**
       - Click sun/moon toggle in top-right nav
       - Verify colors switch to dark palette (deep navy background, sand-gold accents)
       - Refresh page — verify dark mode persists (no flash)
       - Toggle back to light mode
    5. **404 page:**
       - Visit http://localhost:8080/does-not-exist
       - Verify "Lost at Sea" heading, anchor icon, themed copy, "Return to the Bar" button
    6. **Mobile:**
       - Open Chrome DevTools, toggle device toolbar to iPhone size
       - Verify single-column card grid
       - Verify tagline hidden in nav
       - Verify pagination still works
    7. **Typography:**
       - Verify Playfair Display (serif) on site name and post titles
       - Verify Lora (serif) on body text
       - Verify monospace on code blocks
