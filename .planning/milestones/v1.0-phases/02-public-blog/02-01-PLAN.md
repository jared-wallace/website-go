    2. Create `internal/service/post/list.go`:
       - Define `const PostsPerPage = 10` (per D-07)
       - Define `ListResult` struct:
         ```go
         type ListResult struct {
             Posts       []PostSummary
             CurrentPage int
             TotalPages  int
             HasPrev     bool
             HasNext     bool
         }

         type PostSummary struct {
             Title       string
             Slug        string
             Excerpt     string
             Tags        []string
             PublishedAt time.Time
             ReadingTime int
         }
         ```
       - `func (s *Service) ListPublished(ctx context.Context, page int) (ListResult, error)`:
         - Clamp page to >= 1
         - Call `s.repo.CountPublished(ctx)` for total
         - Compute `totalPages = int(math.Ceil(float64(total) / float64(PostsPerPage)))`
         - If totalPages == 0, set totalPages = 1 (empty state still page 1)
         - Clamp page to <= totalPages
         - Compute offset = (page - 1) * PostsPerPage
         - Call `s.repo.ListPublished(ctx, PostsPerPage, offset)`
         - Map each `model.Post` to `PostSummary` with `ReadingTime(p.Body)`, `Excerpt(p.Body, 200)`, `ParseTags(p.Tags)`
         - Return ListResult with pagination metadata

    3. Create `internal/service/post/get.go`:
       - Define `PostDetail` struct:

         ```go
         type PostDetail struct {
             Post        model.Post
             RenderedHTML template.HTML  // cast from string to avoid double-escape (Pitfall 3)

             ReadingTime int
         }
         ```
       - `func (s *Service) GetBySlug(ctx context.Context, slug string) (*PostDetail, error)`:
         - Call `s.repo.FindBySlug(ctx, slug)`
         - If error, return (let handler check for ErrNotFound)
         - Compute ReadingTime, ExtractToC from RenderedHTML, ParseTags
         - Cast RenderedHTML to `template.HTML`
         - Return PostDetail
