```go
func RequireSession(sm *scs.SessionManager) func(http.Handler) http.Handler {
    return func(next http.Handler) http.Handler {
        return http.HandlerFunc(func(w http.ResponseWriter, r *http.Request) {
```

    Create RequireAPIToken following the same shape as RequireSession. The function signature:
    ```go
    func RequireAPIToken(token string) func(http.Handler) http.Handler
    ```
    Implementation:
    1. Return a middleware closure that wraps the next handler
    2. Read `r.Header.Get("Authorization")`
    3. Strip "Bearer " prefix via `strings.TrimPrefix(raw, "Bearer ")`
    4. If `raw == got` (TrimPrefix did nothing, meaning no "Bearer " prefix was present), return 401
    5. Use `crypto/subtle.ConstantTimeCompare([]byte(got), []byte(token))` — prevents timing oracle attacks
    6. If result != 1, return 401 with `http.Error(w, "unauthorized", http.StatusUnauthorized)`
    7. If token config is empty string, always return 401 (endpoint disabled when API_TOKEN not set)
    8. Call `next.ServeHTTP(w, r)` on success

    Import `crypto/subtle` (stdlib) and `strings` (stdlib). No new dependencies.

    ```go
    // UpsertBySlug creates or updates a post identified by slug.
    // New posts land as unpublished drafts (D-11). On update, existing
    // tags are preserved — the admin sets tags from the web UI.
    func (s *Service) UpsertBySlug(ctx context.Context, title, slug, body string) error {
        rendered := s.renderer.Render(body)
        existing, err := s.repo.FindBySlug(ctx, slug)
        if err != nil {
            if errors.Is(err, postrepo.ErrNotFound) {
                _, createErr := s.repo.Create(ctx, model.Post{
                    Title:        title,
                    Slug:         slug,
                    Body:         body,
                    RenderedHTML: string(rendered),
                    Published:    false,
                })
                return createErr
            }
            return err
        }
        return s.repo.Update(ctx, model.Post{
            ID:           existing.ID,
            Title:        title,
            Slug:         slug,
            Body:         body,
            RenderedHTML: string(rendered),
            Tags:         existing.Tags,
        })
    }
    ```
    Add `"errors"` to imports, add `postrepo "github.com/jared-wallace/website-go/internal/repository/post"` if not already imported, add `"github.com/jared-wallace/website-go/internal/model"` if not already imported.

    **API handler package (internal/handler/api/handler.go) per D-09, D-10, D-11:**
    Create new package `api` under `internal/handler/`. This follows the existing `handler/blog/` and `handler/admin/` convention.

    ```go
    package api

    import (
        "fmt"
        "io"
        "net/http"

        postservice "github.com/jared-wallace/website-go/internal/service/post"
    )

    // MetaRenderer extracts YAML front matter alongside rendering.
    type MetaRenderer interface {
        RenderWithMeta(src string) (template.HTML, map[string]interface{})
    }

    // APIHandler serves the API push endpoint.
    type APIHandler struct {
        svc      *postservice.Service
        renderer MetaRenderer
    }

    // New creates an APIHandler.
    func New(svc *postservice.Service, renderer MetaRenderer) *APIHandler {
        return &APIHandler{svc: svc, renderer: renderer}
    }
    ```

    PushPost method:
    1. Read body with `io.ReadAll(io.LimitReader(r.Body, 1<<20))` — 1 MB limit for .md files
    2. Call `h.renderer.RenderWithMeta(string(body))` to get `(_, meta)`
    3. Extract `slug` from `meta["slug"].(string)`. If empty or type assertion fails, return 400 "front matter must include 'slug'" (per Pitfall 6: YAML keys are case-sensitive, must be lowercase)
    4. Extract `title` from `meta["title"].(string)`. If empty, fallback to `title = slug`
    5. Call `h.svc.UpsertBySlug(r.Context(), title, slug, string(body))`
    6. On error, return 500 "failed to upsert post"
    7. On success, return 200 with plain text "ok\n" (`fmt.Fprintln(w, "ok")`)

    Set `Content-Type: text/plain` on the response.
