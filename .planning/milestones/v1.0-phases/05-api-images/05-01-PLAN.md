    **Upload handler in internal/handler/admin/upload.go:**
    Create UploadImage method on *AdminHandler. The handler must:
    1. Call `http.NewResponseController(w)` then `rc.SetWriteDeadline(time.Now().Add(30 * time.Second))` (per D-14)
    2. Call `r.ParseMultipartForm(5 << 20)` — 5 MB max (per D-04). On error return 400 "file too large or bad request"
    3. Call `r.FormFile("image")`. On error return 400
    4. Read first 512 bytes into buffer, call `http.DetectContentType(buf[:n])` (per D-12)
    5. If MIME is not "image/jpeg" and not "image/png", return 415 "only JPEG and PNG accepted" (per D-03)
    6. Call `file.Seek(0, io.SeekStart)` — critical: without this the stored file is truncated by 512 bytes
    7. Generate random filename: `crypto/rand.Read(b)` where b is 16 bytes, then `hex.EncodeToString(b)` + extensionFromMIME(mime). extensionFromMIME returns ".jpg" for "image/jpeg", ".png" for "image/png"
    8. Create destination file at `filepath.Join(h.imageDir, filename)` via `os.Create`
    9. `io.Copy(dst, file)` to write the full file
    10. Return JSON: `{"url": "/images/{filename}", "markdownTag": "![alt](/images/{filename})"}`

    Use `encoding/json` for the response. Set `Content-Type: application/json`.

    **Helper function** (unexported, same file):
    ```go
    func extensionFromMIME(mime string) string {
        switch mime {
        case "image/jpeg": return ".jpg"
        case "image/png": return ".png"
        default: return ""
        }
    }
    ```

    Add the following INSIDE the existing IIFE, after the Mobile Tab Toggle section and before the closing `})();`:

    ```javascript
    // --- Image Upload (D-01, D-02) ---
    var uploadBtn = document.getElementById('upload-image-btn');
    var uploadInput = document.getElementById('upload-image-input');
    var uploadError = document.getElementById('upload-error');

    if (uploadBtn && uploadInput && editorBody) {
      uploadBtn.addEventListener('click', function() {
        uploadInput.click();
      });

      uploadInput.addEventListener('change', function() {
        var file = uploadInput.files[0];
        if (!file) return;

        // Clear previous error
        uploadError.style.display = 'none';
        uploadError.textContent = '';

        // Client-side size check (5 MB per D-04)
        if (file.size > 5 * 1024 * 1024) {
          uploadError.textContent = 'File exceeds 5 MB limit. Choose a smaller image.';
          uploadError.style.display = '';
          uploadInput.value = '';
          return;
        }

        uploadBtn.textContent = 'Uploading\u2026';
        uploadBtn.disabled = true;

        var formData = new FormData();
        formData.append('image', file);

        fetch('/admin/images/upload', { method: 'POST', body: formData })
          .then(function(r) {
            if (r.status === 415) throw new Error('Only JPEG and PNG are accepted.');
            if (r.status === 400) throw new Error('File exceeds 5 MB limit. Choose a smaller image.');
            if (!r.ok) throw new Error('Upload failed. Try again.');
            return r.json();
          })
          .then(function(data) {
            insertAtCursor(editorBody, data.markdownTag);
          })
          .catch(function(err) {
            uploadError.textContent = err.message;
            uploadError.style.display = '';
          })
          .finally(function() {
            uploadBtn.textContent = 'Upload Image';
            uploadBtn.disabled = false;
            uploadInput.value = '';
          });
      });
    }

    function insertAtCursor(textarea, text) {
      var start = textarea.selectionStart;
      var end = textarea.selectionEnd;
      textarea.value = textarea.value.slice(0, start) + text + textarea.value.slice(end);
      textarea.selectionStart = textarea.selectionEnd = start + text.length;
      textarea.dispatchEvent(new Event('input')); // triggers live preview debounce
    }
    ```
