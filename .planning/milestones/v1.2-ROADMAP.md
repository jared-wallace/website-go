## Milestone Summary

**Key Decisions:**
- Background grain uses position: fixed pseudo-element (not background-attachment: fixed) to avoid iOS scroll jank
- Grain z-index: 9999 with pointer-events: none -- above all content but click-through
- Dark mode transitions gated behind .theme-ready to prevent flash-of-transition on page load
- 250ms animation duration chosen as human-perceptible but sub-threshold for 'slow'
- Card stagger capped at 6 cards (375ms max total) to avoid long waits
- Footer About link uses footer-link+nav-link classes to disambiguate from nav-bar pattern in tests
- SVG rope uses two interleaving sinusoidal paths with stroke=var(--color-divider) for automatic dark mode support

**Issues Resolved:**
- Reaction button pill radius inconsistency fixed (2rem -> 4px)
- Stale "The Log" CSS comment rebranded to "The Wild Meridian"
- Pre-existing reaction-bounce animation brought into WCAG 2.3.3 compliance via reduced-motion guard
