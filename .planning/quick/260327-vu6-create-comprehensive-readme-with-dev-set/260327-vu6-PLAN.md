- Step-by-step: clone, copy .env.example to .env (for local dev use localhost not postgres hostname), `make dev-up` (starts Postgres), `make migrate`, `make dev` (hot reload)

- Migrations live in db/migrations/ as sequential SQL files

**9. Testing**
- `make test` (go test with race detector), `make lint` (golangci-lint)
- CI runs via GitHub Actions on push/PR

**10. Docker**
- `make docker` builds multi-stage image (golang:1.26-alpine build, alpine:3.21 runtime)

- Production: docker-compose.yml runs app + Postgres containers

- Deploy steps: SSH to EC2, cd to repo, `make deploy` (pulls, builds, restarts)
- Important notes from STATE.md: ASG max_size=1, EBS delete_on_termination=false, chown 999:999 on pgdata dir before first deploy
