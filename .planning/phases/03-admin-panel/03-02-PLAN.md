    `LoginPage(w, r)`: If already authenticated (`sm.GetBool(ctx, "authenticated")`), redirect to `/admin/posts`. Otherwise render "admin-login.html" with status 200. Accept optional flash error from session via `sm.PopString(ctx, "flash_error")`.

    `LoginPost(w, r)`: Per D-11, D-08, and Pitfall 6:
    1. Parse form: `email := r.FormValue("email")`, `password := r.FormValue("password")`
    2. Check rate limiter: `if !h.rateLimiter.Allow(extractIP(r))` -> render login with 429 and message "Too many attempts. Try again later." (per UI-SPEC copywriting)
    3. Verify credentials in constant-time:
       ```go
       hashToCheck := h.adminHash
       if email != h.adminEmail {
           hashToCheck = h.dummyHash
       }
       err := bcrypt.CompareHashAndPassword(hashToCheck, []byte(password))
       if email != h.adminEmail || err != nil {
           // render login with 401 and "Invalid email or password."
       }
       ```
    4. On success: `sm.RenewToken(r.Context())` then `sm.Put(r.Context(), "authenticated", true)` then redirect 303 to `/admin/posts`

    `Logout(w, r)`: Call `sm.Destroy(r.Context())` then redirect 303 to `/admin/login`.

    Helper `extractIP(r *http.Request) string`: Check `X-Real-IP` header first (Nginx sets this), fall back to `r.RemoteAddr`. Strip port with `net.SplitHostPort`.

    **Host router** in `cmd/server/main.go`:
    ```go
    type hostRouter struct {
        blog  http.Handler
        admin http.Handler
    }
    func (hr *hostRouter) ServeHTTP(w http.ResponseWriter, r *http.Request) {
        host := r.Host
        if h, _, err := net.SplitHostPort(host); err == nil {
            host = h
        }
        if host == cfg.AdminHost {
            hr.admin.ServeHTTP(w, r)
            return
        }
        hr.blog.ServeHTTP(w, r)
    }
    ```
