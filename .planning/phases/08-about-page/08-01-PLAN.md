    **5. Create `internal/handler/blog/about.go`** — the AboutPage handler method:
    ```go
    package blog

    import (
        "net/http"

        "github.com/jared-wallace/website-go/content"
        "github.com/jared-wallace/website-go/internal/markdown"
    )

    // AboutPage handles GET /about, rendering the embedded about.md through
    // the goldmark pipeline into the about.html template.
    func (h *BlogHandler) AboutPage(w http.ResponseWriter, r *http.Request) {
        renderer := markdown.NewRenderer()
        rendered := renderer.Render(content.AboutMD)
        h.render(w, http.StatusOK, "about.html", map[string]interface{}{
            "RenderedHTML": rendered,
        })
    }
    ```

    Note: Creating a new renderer per request is wasteful. Instead, the BlogHandler should accept the renderer or hold a reference. Check if the `svc` (postservice.Service) already exposes the renderer, or add a `renderer` field to BlogHandler. Looking at the handler constructor, it takes `*postservice.Service`. The renderer is created in main.go and passed to the service. The simplest approach: add a `md *markdown.Renderer` field to BlogHandler, update `New()` to accept it, and use it in AboutPage. Update `cmd/server/main.go` accordingly: `blog := bloghandler.New(svc, renderer)`.
