# Phase 2 — UI Design Contract

> Visual and interaction contract for the public reading experience. Generated by gsd-ui-researcher, verified by gsd-ui-checker.

---

## Design System

| Property | Value |
|----------|-------|
| Tool | none — manual CSS (Go html/template, no React/Next.js/Vite) |
| Preset | not applicable |
| Component library | none — custom CSS components |
| Icon library | inline SVG — anchor, sun, moon icons hand-coded; no external icon dependency |
| Font — Display | Playfair Display (Google Fonts) — serif, editorial gravitas, beach club menu energy |
| Font — Body | Lora (Google Fonts) — readable serif, warm and literary, pairs with Playfair without fighting it |

**Rationale (frontend-design skill):** The weathered beach bar aesthetic calls for editorial serif typography. Playfair Display delivers the signage/menu feel of a nautical bar; Lora keeps body copy comfortable for long reads. Both are characterful without being novelty fonts. Deliberately avoiding sans-serif system fonts per the skill's "no generic AI slop" directive.

## Spacing Scale

Declared values (multiples of 4 only):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, tag pill horizontal padding |
| sm | 8px | Intra-card spacing, metadata gaps |
| md | 16px | Card internal padding, paragraph margin |
| lg | 24px | Card grid gap, section padding |
| xl | 32px | Post content max-width side padding |
| 2xl | 48px | Major section breaks (header → grid, post title → body) |
| 3xl | 64px | Page top/bottom padding |

Exceptions:
- Touch targets (dark mode toggle, pagination buttons): minimum 44px tall — WCAG 2.5.5 AA
- ToC toggle button: minimum 44px tall for tap comfort on mobile
- Card hover lift: `transform: translateY(-4px)` — uses 4px, consistent with scale

## Typography

| Role | Family | Size | Weight | Line Height | Usage |
|------|--------|------|--------|-------------|-------|
| Display | Playfair Display | 36px | 700 (bold) | 1.15 | Site name/title in nav header |
| Heading | Playfair Display | 26px | 600 (semibold) | 1.2 | Post titles on cards and single post page |
| Body | Lora | 18px | 400 (regular) | 1.7 | Post body prose, excerpt text |
| Label | Lora | 14px | 400 (regular) | 1.4 | Metadata (date, reading time), tag pills, pagination labels |

**Note on label weight:** Label uses 400 because Lora's regular weight at 14px has sufficient contrast; no need for a third weight. Two declared weights: 400 and 600/700 (display bold is an extension of the semibold intent, same visual role).

**Code blocks:** Monospace stack — `'Courier Prime', 'Courier New', monospace`. Size: 15px. Weight: 400. Background: secondary surface color. Padding: md (16px).

| Role | Hex | Usage |
|------|-----|-------|
| Dominant (60%) | `#F5F0E8` | Page background — warm sand parchment |
| Secondary (30%) | `#E8DFD0` | Card backgrounds, nav bar, footer, code block backgrounds |
| Accent (10%) | `#2C5F7A` | See reserved list below |
| Text primary | `#2C2418` | Body text, headings — dark driftwood brown |
| Text secondary | `#7A6A55` | Metadata labels, tag text, placeholder copy |
| Divider | `#C8B89A` | Rope-style hr dividers, card border |
| Destructive | n/a | No destructive actions in Phase 2 (read-only public view) |

**Accent (`#2C5F7A` — deep ocean blue) reserved for:**
- Post title links on cards (on hover)
- Pagination active page number
- ToC anchor links (on hover)
- Dark mode toggle icon
- In-post anchor links within prose

### Dark Mode ("Night Beach")

| Role | Hex | Usage |
|------|-----|-------|
| Dominant (60%) | `#1A1F2E` | Page background — deep midnight navy |
| Secondary (30%) | `#242B3D` | Card backgrounds, nav bar, footer, code block backgrounds |
| Accent (10%) | `#C9A84C` | Same reserved list as light mode — sand-gold replaces ocean blue |
| Text primary | `#E8DFD0` | Body text, headings — warm sand |
| Text secondary | `#9A8B75` | Metadata, tag text |
| Divider | `#3A4558` | Dividers, card borders |
| Destructive | n/a | Not applicable in Phase 2 |

**Dark mode implementation:** CSS custom properties on `:root` overridden by `[data-theme="dark"]` attribute on `<html>`. JavaScript reads `localStorage` key `theme` on load; falls back to `prefers-color-scheme` media query. Toggle flips `data-theme` and writes to `localStorage`.

## Component Contracts

### Nav Bar

- Full-width, sticky top, secondary surface color background
- Left: site name in Display typography (Playfair Display 36px bold) — links to `/`
- Right: dark mode toggle (sun/moon SVG inline icon, 44px touch target)
- No additional navigation links in Phase 2 (no admin, no other sections yet)
- Bottom border: 1px solid divider color
- Padding: lg (24px) horizontal, sm (8px) vertical

### Post Card

- Background: secondary surface color
- Border: 1px solid divider color
- Border-radius: 4px (subtle, not pill-shaped — weathered wood doesn't round)
- Padding: lg (24px) all sides
- Content order (top to bottom):
  1. Tags row — small pill labels, text-secondary color, xs (4px) gap between pills
  2. Post title — Heading typography, text-primary, no underline at rest
  3. Excerpt — Body typography at 16px (one step down from 18px for scan-friendliness), 3 lines max, `overflow: hidden; display: -webkit-box; -webkit-line-clamp: 3`
  4. Metadata row — Label typography, text-secondary: `{date}  ·  {N} min read`
- Hover state: `transform: translateY(-4px)`, box-shadow `0 8px 24px rgba(44, 36, 24, 0.12)` (light) / `rgba(0,0,0,0.4)` (dark), transition 200ms ease
- Post title links to `/posts/{slug}` — color inherits text-primary at rest, accent on hover
- Card height: uniform via CSS grid `align-items: stretch`; metadata row pinned to bottom via `display: flex; flex-direction: column; justify-content: space-between` on card inner

### Card Grid

- CSS Grid: `grid-template-columns: repeat(2, 1fr)` on desktop (≥ 768px)
- CSS Grid: `grid-template-columns: 1fr` on mobile (< 768px)
- Gap: lg (24px)
- Max-width container: 900px centered, auto side margins

### Tag Pill

- Display: inline-block
- Padding: 2px xs (4px) horizontal — total 4px × 2 + text
- Border: 1px solid divider color
- Border-radius: 2px (near-square, like a stamp)
- Font: Label typography (14px, 400)
- Color: text-secondary
- Background: transparent (inherits card background)
- Not interactive in Phase 2 (no click/hover state beyond cursor: default)

### Post Page Layout

- Max-width: 700px centered (optimal prose line length ~70 characters at 18px Lora)
- Header section: title (Heading 26px), then metadata row (date · reading time), then ToC (if applicable), then `<hr>` divider, then body
- Post title: Playfair Display, 32px, weight 700 on single post page (larger than card heading)

### Table of Contents

- Appears only when post has 3 or more h2/h3 headings (D-14)
- Position: inline block after metadata, before body prose
- Container: secondary surface color background, border 1px solid divider, border-radius 4px, padding md (16px)
- Header row: "Contents" label (Label 14px, weight 600) + toggle button "[ hide ]" / "[ show ]" text button (accent color, no underline, cursor pointer)
- Collapsed state: only header row visible, body hidden via `max-height: 0; overflow: hidden`
- Expanded state: body visible, smooth transition 200ms
- List items: h2 at body indentation; h3 at md (16px) indent
- Link color: accent color, no underline at rest, underline on hover
- Animation: CSS `max-height` transition (no JavaScript animation library)
- Default state: expanded

### Pagination

- Alignment: centered, below card grid, mt-2xl (48px top margin)
- Structure: `< Prev` · `1` `2` `3` · `Next >`
- Active page: accent color text, bold, no link
- Inactive pages: text-secondary, link to `/posts?page=N`
- Prev/Next: text-secondary when at boundaries (cursor: default, no link); accent when navigable
- Touch targets: 44px minimum height, padding sm (8px) horizontal per item
- Separator dots between prev/numbers/next: text-secondary `·`

### 404 Page

- Full page, centered vertically and horizontally
- Anchor icon (SVG, 48px) in accent color at top
- Heading: "Lost at Sea" — Playfair Display, 36px, text-primary
- Body: "This page has gone overboard. The tide may have taken it, or perhaps it never existed." — Lora 18px, text-secondary
- CTA button: "Return to the Bar" — links to `/`; accent background color, dominant-surface text, padding md (16px) × lg (24px), border-radius 4px

### Horizontal Divider (Rope)

- `<hr>` styled as: `border: none; border-top: 2px dashed divider-color; opacity: 0.6`
- This approximates a rope-twist look without SVG complexity
- Used: between post header and body, at footer top

### Code Blocks

- Background: secondary surface color
- Border-left: 3px solid accent color
- Padding: md (16px)
- Font: `'Courier Prime', 'Courier New', monospace`, 15px
- Border-radius: 0 4px 4px 0 (flush left edge, rounded right)
- Overflow: `overflow-x: auto`

## Accessibility Contract

- Color contrast: text-primary on dominant background achieves ≥ 7:1 (AAA) in both light and dark modes
- Color contrast: text-secondary on secondary surface achieves ≥ 4.5:1 (AA) in both modes
- All interactive elements reachable via keyboard Tab
- Focus indicators: 2px solid accent outline, 2px offset — never removed
- Dark mode toggle: aria-label required (see copywriting contract)
- Images (Phase 2 has no images on listing page — n/a)
- ToC: rendered as `<nav aria-label="Table of contents">` wrapping an `<ol>`
- Post article: wrapped in `<article>` element
- Post listing: each card is an `<article>` with heading as first child
